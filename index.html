❓ Help & Information

14 Features Measured

Typing Features (8):

  • Typing Speed (WPM) - Words per minute
  • Error Rate - Proportion of errors
  • Backspace Rate - Correction frequency
  • Burst Duration - Longest pause between keys
  • Sentence Pause - Average thinking pause
  • Key Variance - Typing rhythm stability
  • Correction Latency - Time to fix mistakes
  • Typing Entropy - Pattern randomness

Mouse Features (6):

  • Cursor Speed Mean - Average mouse speed
  • Speed Variance - Control stability
  • Path Efficiency - Movement directness
  • Click Rate - Clicking frequency
  • Hover Time - Decision hesitation
  • Movement Entropy - Direction randomness
